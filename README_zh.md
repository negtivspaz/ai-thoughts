# ai_thoughts

[English](README.md)

文章与随笔合集，围绕两大主题：借助AI工具运营精简的一人企业，以及对 [OpenClaw](https://openclaw.ai)（一款开源AI代理平台）的实战体验。内容涵盖实操指南（自定义技能开发、自托管部署加固），以及更具个人色彩的写作（摩托车文化、个人创业）。

每篇文章的配图存放于 [`imgs/`](imgs/) 子目录，沿用与文章相同的 `YYMMDDHHММ_slug` 命名规范。
